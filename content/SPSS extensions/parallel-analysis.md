SPSS custom dialog for determining the number of components or factors underlying a set of variables using parallel analysis. The program is based on the work of Brian O’Conner (2000) and provides a dialog version of the [“rawpar.sps” syntax](https://oconnor-psych.ok.ubc.ca/nfactors/rawpar.sps).

Download the [SPSS custom dialog, manual and example data](https://github.com/kloosterpm/SPSS-extensions/blob/main/SPSS-Parallel-Analysis.zip) on GitHub (click the `Download raw file` button).
